## Название работы

**Book Tracker**
**(Трекер прочитанных книг)**

## Краткое описание программы

GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации, сохранением данных в JSON и использованием Git.


## Инструкция по запуску (что установить, как запустить)

### Инструкция по запуску Book Tracker

#### Что нужно установить

1.  **Python 3.6 или выше**
    
    -   Перейдите на официальный сайт Python: python.org.
        
    -   Скачайте последнюю версию для вашей ОС (Windows, macOS, Linux).
        
    -   При установке на Windows **обязательно отметьте галочку «Add Python to PATH»**.
        
2.  **Библиотека Tkinter**
    
    -   Обычно идёт в комплекте с Python — отдельно устанавливать не нужно.
        
    -   Чтобы проверить наличие, откройте командную строку (терминал) и выполните:
        
        bash
        
        ```bash
        python -c "import tkinter; print('Tkinter установлен')"
        
        ```
        
    -   Если видите сообщение «Tkinter установлен» — всё в порядке. Если возникает ошибка, переустановите Python с отмеченной опцией установки Tkinter.
        
3.  **Дополнительные библиотеки**
    
    -   Для данного приложения требуются только стандартные библиотеки Python (`tkinter`, `json`, `os`), поэтому дополнительных установок не требуется.
        

----------

#### Как запустить приложение

**Способ 1. Запуск из командной строки (рекомендуется)**

1.  Сохраните полный код приложения в файл с именем `book_tracker.py`.
    
2.  Откройте командную строку или терминал.
    
3.  Перейдите в папку, где сохранён файл `book_tracker.py`, с помощью команды `cd`:
    
    bash
    
    ```bash
    cd путь/к/папке/с/приложением
    
    ```
    
4.  Запустите приложение командой:
    
    bash
    
    ```bash
    python book_tracker.py
    
    ```
    
    -   На Linux/macOS может потребоваться команда `python3 book_tracker.py`.
        

**Способ 2. Двойной клик по файлу**

1.  Сохраните код в файл `book_tracker.py`.
    
2.  Найдите файл в проводнике (Explorer на Windows, Finder на macOS) и дважды щёлкните по нему.
    
3.  Приложение запустится, если Python правильно ассоциирован с файлами `.py`.
    
    -   **Важно:** при таком способе окно консоли может не появиться, и ошибки будут незаметны. Для отладки лучше использовать командную строку.
        

**Способ 3. Запуск из среды разработки (IDE)**

Если вы используете PyCharm, VS Code, IDLE и т. д.:

1.  Создайте новый проект или откройте существующий.
    
2.  Создайте файл `book_tracker.py` и вставьте в него код.
    
3.  Запустите скрипт кнопкой «Run» или сочетанием клавиш (например, `F5` в IDLE).

# Примеры использования


### Примеры использования Book Tracker

#### Пример 1. Добавление книги

**Цель:** внести в трекер новую прочитанную книгу.

**Шаги:**

1.  Запустите приложение.
    
2.  В поле **«Название книги»** введите: _«Мастер и Маргарита»_.
    
3.  В поле **«Автор»** введите: _Михаил Булгаков_.
    
4.  В поле **«Жанр»** введите: _Роман_.
    
5.  В поле **«Количество страниц»** введите: _480_.
    
6.  Нажмите кнопку **«Добавить книгу»**.
    

**Результат:** книга появляется в таблице внизу окна. Поля ввода очищаются, готовы к добавлению следующей книги.

----------

#### Пример 2. Проверка валидации (некорректный ввод)

**Цель:** проверить, как приложение реагирует на ошибки ввода.

**Сценарий А: пустое поле «Название книги»**

1.  Оставьте поле **«Название книги»** пустым.
    
2.  Заполните остальные поля корректными данными.
    
3.  Нажмите **«Добавить книгу»**.
    

**Результат:** появляется окно с сообщением _«Название книги не может быть пустым!»_. Книга не добавляется.

**Сценарий Б: нечисловое значение в «Количество страниц»**

1.  В поле **«Количество страниц»** введите текст: _четыреста_.
    
2.  Заполните остальные поля.
    
3.  Нажмите **«Добавить книгу»**.
    

**Результат:** появляется окно с сообщением _«Количество страниц должно быть положительным целым числом!»_. Книга не добавляется.

----------

#### Пример 3. Фильтрация книг

**Цель:** отфильтровать книги по жанру и объёму.

**Шаг 1. Подготовка данных**  
Добавьте несколько книг:

-   _«1984»_, Джордж Оруэлл, _Антиутопия_, 328 стр.
    
-   _«Гарри Поттер и философский камень»_, Дж. К. Роулинг, _Фэнтези_, 223 стр.
    
-   _«Преступление и наказание»_, Ф. М. Достоевский, _Роман_, 671 стр.
    
-   _«Маленький принц»_, Антуан де Сент‑Экзюпери, _Сказка_, 96 стр.
    

**Шаг 2. Фильтр по жанру**

1.  В поле **«Фильтр по жанру»** введите: _Фэнтези_.
    
2.  Нажмите **«Применить фильтр»**.
    

**Результат:** в таблице остаётся только книга _«Гарри Поттер и философский камень»_. Остальные скрыты.

**Шаг 3. Фильтр по количеству страниц**

1.  Очистите поле фильтра по жанру.
    
2.  В поле **«Минимум страниц»** введите: _300_.
    
3.  Нажмите **«Применить фильтр»**.
    

**Результат:** в таблице отображаются книги _«1984»_ (328 стр.) и _«Преступление и наказание»_ (671 стр.).

**Шаг 4. Комбинированный фильтр**

1.  В поле **«Фильтр по жанру»** введите: _Роман_.
    
2.  В поле **«Минимум страниц»** оставьте _300_.
    
3.  Нажмите **«Применить фильтр»**.
    

**Результат:** отображается только _«Преступление и наказание»_.

**Шаг 5. Сброс фильтра**

1.  Нажмите кнопку **«Сбросить фильтр»**.
    

**Результат:** в таблице снова видны все добавленные книги.

----------

#### Пример 4. Сохранение и загрузка данных

**Цель:** сохранить список книг и восстановить его после перезапуска приложения.

**Шаг 1. Сохранение**

1.  Добавьте 2–3 книги (как в примере 3).
    
2.  Нажмите кнопку **«Сохранить в JSON»**.
    

**Результат:**

-   В папке с приложением создаётся файл `books.json`.
    
-   Появляется сообщение _«Данные сохранены в books.json»_.
    
-   Содержимое `books.json` (пример):
    
    json
    
    ```json
    [
      {
        "title": "1984",
        "author": "Джордж Оруэлл",
        "genre": "Антиутопия",
        "pages": 328
      },
      {
        "title": "Преступление и наказание",
        "author": "Ф. М. Достоевский",
        "genre": "Роман",
        "pages": 671
      }
    ]
    
    ```
    

**Шаг 2. Загрузка**

1.  Закройте приложение.
    
2.  Снова запустите `book_tracker.py`.
    
3.  Нажмите кнопку **«Загрузить из JSON»**.
    

**Результат:** все книги из `books.json` отображаются в таблице.

----------

#### Пример 5. Полный цикл работы

**Цель:** смоделировать типичное использование приложения за один сеанс.

1.  **Запуск:** откройте `book_tracker.py`. Автоматически загружаются книги из `books.json` (если файл есть).
    
2.  **Добавление:** внесите 2 новые книги, убедившись, что валидация работает.
    
3.  **Фильтрация:** отфильтруйте книги объёмом более 200 страниц.
    
4.  **Сохранение:** нажмите **«Сохранить в JSON»**, чтобы зафиксировать изменения.
    
5.  **Завершение:** закройте приложение. При следующем запуске ваши данные будут доступны.
